What is a browser?

A web browser is software that retrieves, presents, and navigates information on the World Wide Web. It renders HTML, CSS, and executes JavaScript so users can view and interact with web pages.

Types of browsers (categories and examples):

- Mainstream graphical browsers: Google Chrome, Mozilla Firefox, Apple Safari, Microsoft Edge, Opera
- Mobile browsers: Chrome for Android, Safari on iOS, Samsung Internet, UC Browser
- Privacy/security-focused browsers: Brave, Tor Browser, Firefox Focus
- Lightweight/minimal browsers: Midori, Falkon, Pale Moon, Lynx (text-mode)
- Text-based/terminal browsers: Lynx, w3m, Links
- Headless browsers (for automation/testing): Headless Chrome, Puppeteer, Playwright, PhantomJS (legacy)
- Kiosk/embedded browsers: WebView (Android), WKWebView (iOS), CEF (Chromium Embedded Framework)
- Specialized browsers: gaming-console browsers, smart-TV browsers, browser-based readers (e.g., RSS clients with rendering)
