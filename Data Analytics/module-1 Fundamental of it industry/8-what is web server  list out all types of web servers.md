# What is a Web Server? 

A web server is a computer or software that stores websites and sends web pages to users when they request them through a web browser.

A web server is a computer or software that stores website files and delivers them to users over the Internet.

** Example **
When you open www.google.com, the web server sends the Google web page to your browser.


# Types of web Server

| Type of Web Server | Developed By               | Main Use                     |
| ------------------ | -------------------------- | ---------------------------- |
| Apache HTTP Server | Apache Software Foundation | General websites             |
| Nginx              | Nginx, Inc.                | High-traffic websites        |
| Microsoft IIS      | Microsoft                  | Windows and ASP.NET websites |
| LiteSpeed          | LiteSpeed Technologies     | Fast web hosting             |
| Lighttpd           | Lighttpd Project           | Lightweight web server       |
| Apache Tomcat      | Apache Software Foundation | Java web applications        |
