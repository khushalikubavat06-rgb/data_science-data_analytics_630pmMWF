# What is a Website ? 

A website is a collection of related web pages that are connected together and available on the Internet under one domain name.

A website is a collection of web pages that provides information or services on the Internet.

** Example  **

1. google.com
2. wikipedia.org
3. amazon.in
4. youtube.com
